# Financial Modeling Lab (Starter)

A beginner-friendly financial modeling app built with Python and Streamlit.

## Features
- Interactive 3-statement-lite projection (revenue, costs, free cash flow)
- Tooltips on every important input
- Dedicated formulas and definitions page
- Built to grow over time with additional models

## Quick Start

1. Create and activate a virtual environment:

```bash
python3 -m venv .venv
source .venv/bin/activate
```

2. Install dependencies:

```bash
pip install -r requirements.txt
```

3. Run the app:

```bash
streamlit run Home.py
```

## Starter Model Included
- Revenue growth projection
- Gross margin and operating expense assumptions
- EBITDA and Free Cash Flow estimation
- Discounted cash flow (simple enterprise value estimate)
